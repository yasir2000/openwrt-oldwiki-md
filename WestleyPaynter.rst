=== Notes about TFTP ===

TFTP commands that I use that are useful. (using tftp in Back|Track linux livecd)
{{{tftp 192.168.1.1
binary
rexmt 1
trace
put firmware.bin}}}
Also sometimes it is helpful to set the session timeout high. (5000 seconds)

{{{
timeout 5000
}}}

= 24XLGi =
The OSBRiDGE 24XLGi is a router used for long wireless links

== Hardware ==
The 24XLGi uses a Realtek RTL8186 SoC

{{{
SoC: RTL8186
CPU: LX5280
Switch: RTL8305SC
Ram: 16 MB (2x8MB)
Wireless:
}}}

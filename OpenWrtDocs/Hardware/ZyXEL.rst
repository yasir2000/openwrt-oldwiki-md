== Getting Information ==
The instructions to get information about your device may vary. Basic info with RAM and Flash size for P-600 series can be acquired by using "sys atsh" command from administrator console. To get further details a serial port connection is needed as described at http://www.ixo.de/info/zyxel_uclinux/
